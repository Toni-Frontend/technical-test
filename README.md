## Getting Started

First,
Clone the repository on your computer

After:
run command to install all dependencies
```bash
npm install
``` 

Finally:

run the development server
```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
